# 开始

# 前言

本项目是基于 form-making 进行的二次开发
遵循代码开源并免费提供使用的原则。

# 二次开发了什么？

1、原版表单设计器生成的代码为一个单独的html页面，而二次开发的代码可以生成一个Vue表单组件，随时随地的在你的Vue工程项目中添加一个Vue组件
2、增加实时预览功能，原版表单设计器需要绘制好表单之后再生成预览，二次开发之后的表单设计器可以实时预览表单，有利于栅格布局，增加了易用性
3、未完待续

# 起步

注意事项

请确保您的项目是通过vue-cli工程搭建的，确保node环境

1、安装依赖

//项目根目录终端

$ npm install form-making -S

$ npm install element-ui -S

2、引入依赖

//main.js

//此步骤为引入element-ui 若您项目已经引入 则跳过
import Element from 'element-ui';
Vue.use(Element, { size: 'mini' });
import 'element-ui/lib/theme-chalk/index.css';

import FormMaking from 'form-making'
import 'form-making/dist/FormMaking.css'
Vue.use(FormMaking)

3、新建表单组件

新建一个Vue文件，啥都不需要做
